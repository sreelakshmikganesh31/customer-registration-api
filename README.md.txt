# Customer Registration API

This project is a Django REST Framework (DRF) CRUD API for managing customer data with Swagger & Redoc API documentation.

## Features
- Create, Read, Update, Delete (CRUD) customers
- Swagger UI for API testing
- SQLite database for easy setup
